<h2>Crie uma nova aplicação com o comando:</h2>

~~~
npx create-react-app testes-react
~~~

<h2>Para rodar os testes vá para a pasta src , e execute:</h2>

~~~
npm test
~~~

~~~
test('', () => {
  //Acessar os elementos na tela

  //Interagir com eles
  
  //Fazer seu teste

});
~~~
